## Executive Summary

Both sites serve complementary functions for Tim Montague's Clean Power brand. **cleanpower.group** is a Wix-based consulting/services site; **cleanpowerhour.com** is a Podpage-based podcast site with 340+ episodes. The merger should consolidate authority under cleanpower.group while preserving the substantial SEO equity in cleanpowerhour.com's episode pages.

### Key Findings

- **cleanpowerhour.com has significantly stronger SEO fundamentals** than cleanpower.group: richer structured data (PodcastSeries, Organization, WebSite, BreadcrumbList schemas), better content volume (340+ episodes, 2 blog posts, 20+ reviews), and more descriptive title tags.
- **cleanpower.group has critical SEO deficiencies**: Wix's client-side rendering limits crawlability, meta descriptions are not reliably served, heading hierarchy is inconsistent, and the blog has only 3 posts with minimal engagement.
- **cleanpowerhour.com is missing basic technical SEO files**: no sitemap.xml, no robots.txt (both return 404).
- **Neither site is fully mobile responsive** at 375px viewport width.
- **The redirect map is critical**: cleanpowerhour.com has 340+ indexed episode pages that each carry unique long-tail keyword authority.

## Site 1: cleanpower.group

**Platform:** Wix
**Primary Purpose:** Consulting services, coaching, booking

### Technical SEO

#### Sitemap
- **Status:** Present at `https://www.cleanpower.group/sitemap.xml`
- **Structure:** Sitemap index with 4 child sitemaps:
  - `pages-sitemap.xml` — 25 URLs (last modified: 2026-02-12)
  - `blog-posts-sitemap.xml` — 9 URLs (last modified: 2025-04-08)
  - `booking-services-sitemap.xml` — 5 URLs (last modified: 2024-11-08)
  - `blog-categories-sitemap.xml` — 1 URL (last modified: 2025-04-08)
- **Total indexed URLs:** ~40
- **Issue:** Auto-generated by Wix. Includes low-value pages like `/copy-of-events`, `/copy-of-home`, `/mediakitold`, `/mediakit23` that should be excluded or noindexed.

#### Robots.txt
- **Status:** Present
- **Contents:**
  - Allows all user agents; blocks `lightbox=` parameters
  - Blocks PetalBot entirely
  - 10-second crawl delay on Dotbot and AhrefsBot
  - References sitemap correctly
- **Issue:** Auto-generated by Wix ("go to SEO Tools > Robots.txt Editor to change this"). Should be customized post-merger.

#### HTTPS
- **Status:** PASS — Site loads over HTTPS with valid certificate.

#### Mobile Responsiveness
- **Status:** FAIL — At 375px viewport width, the desktop navigation bar does NOT collapse into a hamburger menu. All nav items (Home, C&I, About, Services, Book a Meeting, Podcast, Blog) remain displayed horizontally, causing potential overflow. This is a significant mobile UX and Core Web Vitals issue.

#### URL Structure
- **Status:** MIXED
- **Good:** Clean, flat URLs without parameters (e.g., `/about`, `/services`, `/podcast`, `/blog`)
- **Bad:** Several duplicate/legacy pages polluting the URL space:
  - `/copy-of-events` (duplicate)
  - `/copy-of-home` (duplicate)
  - `/mediakitold` (deprecated)
  - `/mediakit23` (year-specific, should redirect)
  - `/mediakit25` (year-specific)
  - `/mediakit` (current?)
  - `/bookameeting` vs `/book-online` (redundant booking pages)
- **Issue:** URL slugs like `/cleanpower`, `/cleantech`, `/strategy` are vague and not keyword-optimized.

#### Canonical Tags
- **Status:** Not reliably detected in page source due to Wix's client-side rendering. Wix typically auto-generates canonical tags, but verification is limited by JS rendering.

#### 404 Handling
- **Status:** PASS — Returns proper "404 Error: Page Not Found" page with title "404 Error: Page Not Found".

#### Page Load
- **Status:** CONCERN — Wix sites carry significant JavaScript overhead. The homepage relies on client-side rendering, which impacts:
  - First Contentful Paint (FCP)
  - Largest Contentful Paint (LCP)
  - Search engine crawling (Googlebot renders JS but with delays)

## Merged-Site SEO Strategy

### Recommended Domain: cleanpower.group

The consulting site should be the primary domain, with all podcast content migrated under `/podcast/` as a subdirectory.

### Priority Actions (Pre-Merger)

#### 1. Migrate Off Wix
Wix's client-side rendering is a fundamental SEO limitation. The merged site should use a platform with server-side rendering (Next.js, WordPress, Webflow, or similar) to ensure:
- Reliable meta tag delivery
- Proper heading hierarchy
- Faster page loads
- Full crawlability without JS rendering

#### 2. Replicate Structured Data
The merged site MUST implement at minimum:
- **Organization** schema (from cleanpowerhour.com)
- **LocalBusiness** schema (from cleanpower.group)
- **WebSite** schema with SearchAction
- **PodcastSeries** schema on /podcast
- **PodcastEpisode** schema on each episode page
- **BreadcrumbList** on all pages
- **Person** schema for Tim Montague
- **Service** schema on service pages

#### 3. Preserve Episode Content
Each of the 340+ episodes should have its own page on the merged site at `cleanpower.group/podcast/[episode-slug]` with:
- Full show notes
- Full transcript
- Audio embed
- Guest links
- Proper PodcastEpisode schema

#### 4. Fix Title Tags Across All Pages
Every page needs a unique, descriptive title under 60 characters:

| Page | Current Title | Recommended Title |
|------|---------------|-------------------|
| Homepage | Solar Company Growth Strategies \| Solar Business Coach & Experts | Clean Power Consulting Group \| Cleantech AI Advisory |
| About | About \| Clean Power Consulting Group | About Tim Montague \| Cleantech AI Consultant |
| Services | Commercial Solar Business Consultant \| Solar Sales & Marketing | Cleantech Consulting Services \| Clean Power Group |
| Podcast | Clean Power Hour Podcast \| Solar Industry \| Clean Power | Clean Power Hour Podcast \| Solar & Storage News |
| Blog | Blog \| Clean Power Consulting Group | Clean Energy Blog \| Clean Power Consulting Group |
| C&I Strategy | C&I \| Clean Power Group | C&I Solar Strategy Call \| Scale Commercial Solar |
| Solar Consulting | (too long - 82 chars) | Solar Business Consulting \| Clean Power Group |
| Career Coaching | (too long - 81 chars) | Clean Energy Career Coaching \| Clean Power Group |
| Executive Coaching | (too long - 84 chars) | Executive Coaching for Cleantech Leaders |
| Contact | Contact \| Clean Power Consulting Group | Contact Clean Power Consulting Group |
| Episodes | Episodes \| Clean Power Hour | All Episodes \| Clean Power Hour Podcast |
| Reviews | Listener Reviews \| Clean Power Hour | Podcast Reviews \| Clean Power Hour |
| Training | Training for renewable energy and solar professionals | (Keep - already good) |
| Sponsors | Sponsors \| Clean Power Hour | Podcast Sponsors & Partners \| Clean Power Hour |

#### 5. Fix Heading Hierarchy
- Every page should have exactly ONE H1 tag
- H2s should be used for major sections
- H3s for subsections
- No empty heading tags
- Homepage currently has 10+ H2s; reduce and restructure

#### 6. Create Proper Sitemap and Robots.txt
- Generate a comprehensive XML sitemap including all pages and all episode pages
- Create a robots.txt that references the sitemap and blocks low-value pages

#### 7. Clean Up Duplicate/Legacy Pages
Before migration, remove or redirect:
- `/copy-of-events` -> `/events`
- `/copy-of-home` -> `/`
- `/mediakitold` -> `/media-kit`
- `/mediakit23` -> `/media-kit`
- `/mediakit25` -> `/media-kit`
- Consolidate `/bookameeting` and `/book-online` into one booking page

#### 8. Implement Proper Mobile Navigation
Both sites fail mobile responsiveness tests. The merged site needs a collapsible hamburger menu at mobile breakpoints (below 768px).

#### 9. Content Strategy
- **Blog:** Commit to at least 2 posts/month on the merged site
- **Episode Pages:** Continue weekly podcast with full transcripts
- **Pillar Content:** Create comprehensive guide pages for key topics:
  - "Complete Guide to Commercial Solar Development"
  - "C&I Solar Sales Process Guide"
  - "Energy Storage for Commercial Properties"
  - "FEOC Compliance Guide for Solar Developers"
- **Internal Linking:** Build topic clusters linking blog posts, episodes, and service pages

#### 10. Local SEO
- Maintain and enhance Google Business Profile
- Ensure NAP consistency across all platforms
- Add LocalBusiness schema to the merged site
